# 🛋️ 東京の部屋 レイアウト検討ツール

同居人が増える東京の部屋(1DK・南向き)のレイアウトを検討するための専用Webアプリです。起動すると間取り図から書き起こした部屋のモデル(壁・ドア・窓・キッチン等の固定設備)が最初から入っており、家具を置いて複数のレイアウト案を作り、2D/3Dで比較できます。

ビルド不要・サーバ不要のバニラJS製（Three.jsのみ同梱）。データはブラウザのlocalStorageに自動保存され、JSONファイルとして書き出し・読み込みもできます。

## 専用機能

- **東京の部屋プリセット**: 初回起動で自動読み込み。実測値入りの間取り図から作成
  (居室幅2,940 / 窓〜水回り5,100 / 水回りブロック2,140×1,730 /
  掃き出し窓2,275 / 玄関ドア800 ほか、すべて実測)。
  さらに直したい場合は「📐 実寸補正」や壁・開口の直接編集で調整できる
- **レイアウト案**: 部屋は共通のまま、家具配置だけを「案A」「案B」…として
  複製・切り替え・比較できる
- **所有者タグ**: 家具ごとに 自分/同居人/共用 を設定し、2Dに色付きの目印
  (青=自分、紫=同居人)を表示
- **固定設備**: キッチン・冷蔵庫置場・柱(540角・天井まで)・南窓沿いの
  造り付けベンチ(高さ40cm)をグレー破線表示でドラッグ不可(誤操作防止)

汎用機能(間取り図画像の下敷き・壁の自動検出・手動の壁編集など)もそのまま残っているので、部屋のモデル自体の微修正も可能です。

## 起動方法

ES Modulesを使っているため、ローカルHTTPサーバ経由で開きます:

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

（VS Codeの Live Server や `npx serve` でも可）

GitHub Pages（Settings → Pages → Deploy from branch → `main`）を有効にすれば、そのままWebで公開・利用できます。

## 使い方の流れ

1. **間取り図を読み込む** — 左パネル「🖼 画像を読み込む」で間取り図の写真やスクリーンショットを下敷きとして表示
2. **壁を自動検出** — 「✨ 壁を自動検出」を押すと、画像から太い線を壁として検出しモデルを自動生成。余分な壁は選択してDelete、位置は端点ドラッグで微調整（うまく検出できない画像は「🧱 壁を描く」で手動でなぞる）
3. **縮尺を合わせる** — 「📏 縮尺合わせ」で図面上の寸法がわかる2点（例: 「3640」と書かれた壁の両端）をクリックし、実寸をcmで入力。検出済みの壁も一緒に正しいサイズへ拡縮される
4. **ドア・窓を置く** — 「🚪 ドア」「🪟 窓」で壁をクリック。選択して幅・高さ・床からの高さを調整
5. **家具を置く** — 左の家具カタログから追加し、ドラッグで移動、ハンドルまたはRキーで回転。右パネルで実物のサイズ(cm)や色に合わせる
6. **3Dで確認** — 上部「🧊 3Dビュー」タブへ。ドラッグで回転、ホイールでズーム。「👁 室内」で室内視点、「📷 画像保存」でスクリーンショット
7. **写真を貼る** — 3Dビュー右パネルで部屋の写真を追加し、床・壁に割り当て。床材や壁紙が写った部分をトリミングした写真だときれいに見えます

## 操作メモ

| 操作 | 方法 |
|---|---|
| 画面の移動 | 何もない所をドラッグ / Space+ドラッグ / 中ボタン |
| ズーム | ホイール / 右下の＋− |
| 削除 | 選択して Delete |
| 回転 | R キー（Shift+R で逆回転）/ 緑のハンドルをドラッグ |
| 元に戻す / やり直す | Ctrl+Z / Ctrl+Shift+Z |

## データについて

- 自動保存: localStorage（同じブラウザで開き直せば続きから）
- 「💾 書き出し」でJSONファイルに保存（間取り図・写真も含む）。別ブラウザ・別PCへは「📂 読み込み」で移行
- 写真は保存容量を抑えるため長辺1024px（間取り図は1600px)に縮小して保持

## 技術構成

- バニラ JavaScript (ES Modules)、ビルドなし
- [Three.js](https://threejs.org/) r164（`vendor/` に同梱、importmapで解決）
- 2Dエディタは Canvas 2D API
- 壁はドア・窓の開口をくり抜いた直方体の集合として3D化
- 壁の自動検出はブラウザ内の画像処理のみで実装（外部API不使用）:
  適応二値化（照明ムラのある写真に対応、失敗時は大津の方法にフォールバック）→
  背景・紙のフチなど壁ではありえない太さの塊を除去 →
  クロージングで二重線・破線を融合 → 距離変換で壁の太さを推定 →
  オープニングで文字・寸法線など細い要素を除去 → 水平/垂直バンド抽出で線分化 →
  塗り率フィルタ・平行線マージ・角スナップで整形。
  太い単線・二重線どちらの作図スタイルにも対応。斜め壁は対象外（手動でなぞる）

### 写真からの検出のコツ

- できるだけ真上から、図面が水平・垂直になるように撮る。傾きは「傾き」スライダーで±15°まで補正可能
- 照明の映り込み（グレア）が壁に重なると、その壁がそこで分断される。
  分断された壁は端点ドラッグでつなぐか、映り込みを避けて撮り直す
- 図面外の要素（共用廊下・EVホールなど）の壁も検出される。不要なら選択してDelete
