# 🛋️ 部屋レイアウト 3D シミュレーター

間取り図の画像を下敷きに壁をなぞると部屋の3Dモデルができ、家具を置いて模様替えのレイアウトを検討できるWebアプリです。部屋の写真を床・壁のテクスチャとして貼り付けて、実際の雰囲気に近づけられます。

ビルド不要・サーバ不要のバニラJS製（Three.jsのみ同梱）。データはブラウザのlocalStorageに自動保存され、JSONファイルとして書き出し・読み込みもできます。

## 起動方法

ES Modulesを使っているため、ローカルHTTPサーバ経由で開きます:

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

（VS Codeの Live Server や `npx serve` でも可）

GitHub Pages（Settings → Pages → Deploy from branch → `main`）を有効にすれば、そのままWebで公開・利用できます。

## 使い方の流れ

1. **間取り図を読み込む** — 左パネル「🖼 画像を読み込む」で間取り図の写真やスクリーンショットを下敷きとして表示
2. **縮尺を合わせる** — 「📏 縮尺合わせ」で図面上の寸法がわかる2点（例: 「3640」と書かれた壁の両端）をクリックし、実寸をcmで入力
3. **壁をなぞる** — 「🧱 壁を描く」で図面の壁を順にクリックしてなぞる（ダブルクリック/Escで終了。自動で直角スナップ、Shiftで解除）
4. **ドア・窓を置く** — 「🚪 ドア」「🪟 窓」で壁をクリック。選択して幅・高さ・床からの高さを調整
5. **家具を置く** — 左の家具カタログから追加し、ドラッグで移動、ハンドルまたはRキーで回転。右パネルで実物のサイズ(cm)や色に合わせる
6. **3Dで確認** — 上部「🧊 3Dビュー」タブへ。ドラッグで回転、ホイールでズーム。「👁 室内」で室内視点、「📷 画像保存」でスクリーンショット
7. **写真を貼る** — 3Dビュー右パネルで部屋の写真を追加し、床・壁に割り当て。床材や壁紙が写った部分をトリミングした写真だときれいに見えます

## 操作メモ

| 操作 | 方法 |
|---|---|
| 画面の移動 | 何もない所をドラッグ / Space+ドラッグ / 中ボタン |
| ズーム | ホイール / 右下の＋− |
| 削除 | 選択して Delete |
| 回転 | R キー（Shift+R で逆回転）/ 緑のハンドルをドラッグ |
| 元に戻す / やり直す | Ctrl+Z / Ctrl+Shift+Z |

## データについて

- 自動保存: localStorage（同じブラウザで開き直せば続きから）
- 「💾 書き出し」でJSONファイルに保存（間取り図・写真も含む）。別ブラウザ・別PCへは「📂 読み込み」で移行
- 写真は保存容量を抑えるため長辺1024px（間取り図は1600px)に縮小して保持

## 技術構成

- バニラ JavaScript (ES Modules)、ビルドなし
- [Three.js](https://threejs.org/) r164（`vendor/` に同梱、importmapで解決）
- 2Dエディタは Canvas 2D API
- 壁はドア・窓の開口をくり抜いた直方体の集合として3D化
